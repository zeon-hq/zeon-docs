## Requesting Features

If you have a feature request, [please create an issue on GitHub](https://github.com/zeon-hq/zeon/issues) or post it on our [Discord Community](https://discord.gg/ZtPdPmu9xW). When requesting features, please include the following information:

1. A short, descriptive title.
2. A detailed description of the feature.
3. Why you think the feature should be added.
4. Any other information you think is relevant.

We will always monitor and track feature requests from all channels and document them in our [Roadmap](https://zeon.ducalis.io/ducalis-roadmap/) as a unified reference for everyone.

## Reporting Bugs

If you find a bug in Zeon, [please create an issue on GitHub](https://github.com/zeon-hq/zeon/issues) or post it on our [Discord Community](https://discord.gg/ZtPdPmu9xW). When reporting bugs, please include the following information:

1. A short, descriptive title.
2. A detailed description of the bug, including any output from the console.
3. Steps to reproduce the bug.
4. Expected behavior.
5. Actual behavior.
6. Your browser and operating system ( optional ).
7. Any other information you think is relevant.

We will always monitor and track bug requests from all channels and document them as issues in our [Main Zeon Repository](https://github.com/zeon-hq/zeon/issues) as a unified reference for everyone.
