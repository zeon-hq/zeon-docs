## Requirements

- [Node.js](https://nodejs.org/en/) (v14 or higher)
- [MongoDB](https://www.mongodb.com/)
- [Redis](https://redis.io/)

## Installation

1. Clone the repository

```bash
git clone
```

2. Install dependencies

```bash
npm install
```

3. Copy the `.env.example` file to `.env` and fill in the values

4. Start the bot

```bash
npm start
```
