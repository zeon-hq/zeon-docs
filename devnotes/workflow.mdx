For issues to work on, please see our [GitHub issues](https://github.com/zeon-hq/zeon/issues). We have a number of issues labeled `good first issue` that are great for new contributors.

If you want to work on an issue, please comment on it so we can assign it to you. If you have an idea for a new feature, please create a new issue and describe your idea.

Label Description:
1. `good first issue` - issues that are great for new contributors
2. `help wanted` - issues that we need help with
3. `bug` - issues that are bugs
4. `enhancement` - issues that are enhancements to existing features
5. `feature` - issues that are new features
6. `documentation` - issues that are related to documentation
7. `question` - issues that are questions
8. `invalid` - issues that are invalid
9. `wontfix` - issues that we won't fix
