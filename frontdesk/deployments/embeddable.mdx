The embeddable support chat component let's you add a chat component to your website or application with no-code. This is a great way to gather user feedback and answer queries while still staying right on your application or website's page.

## How to add a chat widget to your website

1. Go to the support chat channel settings you want to create a embeddable widget for.
2. Click on the **Deployments** tab.
3. Copy the **Embedded Chat** script.
4. Paste the script in your in your website or application's code where you would like to embed.
